# Design Asset Memory

This folder is the early home for reusable design and asset memory.

It is intentionally lightweight. Add entries here when a pattern, visual asset, workflow fragment, or reusable direction becomes useful enough to retrieve later.

Suggested entry shape:

- name
- purpose
- source page or workflow
- reuse notes
- public/private posture if relevant
- implementation reference

This is not a DAM, Figma replacement, or runtime asset service.
