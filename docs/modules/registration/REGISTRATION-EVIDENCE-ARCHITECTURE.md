# Registration Evidence Architecture

Initial evidence types:

- Split Sheet
- Signed Composer Split Confirmation
- Performer Declaration
- Performer Session Log
- Master Ownership Agreement
- Deed of Assignment
- Cue Sheet
- Deposit Certificate
- ISRC Documentation
- ISWC Documentation
- Amendment Evidence
- Dispute Evidence

Evidence must support verification, signatures, superseding, expiry where applicable, and linkage to readiness rules.
