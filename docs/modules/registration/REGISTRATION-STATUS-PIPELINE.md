# Registration Status Pipeline

Initial statuses:

- Draft
- Metadata Incomplete
- Evidence Pending
- Split Confirmation Pending
- Performer Confirmation Pending
- Undocumented
- Ready For Submission
- Submitted
- Under Review
- Registered
- Rejected
- Amendment Required
- Disputed
- Superseded
- Withdrawn

## Notes

Undocumented is a critical cashflow-risk state and must block royalty movement to payable lines until resolved.
