# UI Patterns

This folder is reserved for reusable Sentry Sound UI patterns.

Use it when a frontend pattern has proven useful across product surfaces and should be retrieved/adapted rather than rebuilt from scratch.

Early candidates:

- workflow steppers
- right-rail summary panels
- section cards
- locked/unlocked state panels
- file/category cards
- save/review action rows

Do not move active page code here prematurely.
