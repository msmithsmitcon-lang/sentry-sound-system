# Sentry Sound Academy Programme-to-Module Map V1

## Purpose

This document maps academy programmes to their related modules.

It defines:
- module grouping
- shared modules
- prerequisite logic
- progression flow
- platform-thread placement

This is not yet lesson development.

---

# Core Rule

Modules may appear in more than one programme where appropriate.

Shared modules must keep the same canonical structure, outcomes, SLB logic, metadata logic, and standards placeholders.
