# Sentry Sound System Overview

Sentry Sound is a standalone music production, catalogue, licensing and rights-management system.

It is completely separate from StudyEdge.

The only permitted linkage between Sentry Sound and StudyEdge is:

1. Contractual licence agreement
2. API-based music usage
3. Usage counter / usage log
4. Approved asset references

Sentry Sound owns and manages:
- musical works
- recordings / masters
- lyrics
- stems
- artwork
- music videos
- rights registrations
- licensing records
- client usage logs

StudyEdge may consume approved music through an API, but does not own or store the Sentry Sound catalogue.
