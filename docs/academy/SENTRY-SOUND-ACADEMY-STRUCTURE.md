# Sentry Sound Academy Structure

## Core Separation

### Sentry Sound Platform
The Sentry Sound Platform is the SaaS operational system for:
- compliance
- registrations
- rights tracking
- royalties
- metadata
- contracts
- workflows
- governance
- auditability

### Sentry Sound Academy
The Sentry Sound Academy is a separate training institution/business.

It teaches music industry, technical, business, compliance, creative, and platform-operation competencies.

The platform is not the academy.

The platform becomes one practical module/domain inside the academy.

## Development Rule

Do not develop lessons yet.

First define:
1. Academy domains
2. Module list
3. Competency levels
4. Standards alignment
5. Metadata alignment
6. Canonical module blueprint
7. Then lesson and SLB structures

---

## Backend Alignment Rule

Academy modules must not invent operational workflows that conflict with the Sentry Sound Platform backend.

Where academy content teaches platform operations, it must align to the actual platform logic, entities, workflows, compliance structures, and metadata fields.

Backend changes should only be recommended when they are critical.
