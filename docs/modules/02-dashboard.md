# Dashboard UI Direction

The Sentry Sound dashboard will follow the supplied dark mockup design.

## Current connected elements
- Total Songs tile reads from musical_works.
- Recent Songs reads from musical_works.
- Create Song links to /create-song.
- View Songs links to /list-songs.

## Dead / placeholder elements for now
- Total Streams
- Total Royalties
- Active Projects
- File Vault
- Calendar
- Recent Activity
- Upload Files
- New Project
- Invite Artist

## Build rule
UI can be designed ahead of logic, but connected data must be wired where tables already exist.
